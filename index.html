All Commands

R.help - see all bot comands
R.botinfo - shows some information about the bot
R.ping - shows the bot's ping
R.prefix - add a new prefix (you can only have 2 prefix default one and the custom one)
R.reset-prefix - reset the bot's custom prefix
R.avatar - get your own avatar or the avatar if another user
R.server-info - displays detailed information about the server
R.userinfo - Shows some information about the user
R.profile - this command allows you to view a user's profile using an image
R.membercount - displays the member count of the server
R.server-icon - displays the server icon

R.ban - ban a user from the server
R.unban - unban a user from the server
R.kick - kick a user from the server
R.lock - lock the current channel or the mentioned channel
R.unlock - unlock the current channel or the mentioned channel
R.nuke - deletes and recreates the current channel
R.purge - Delete a specified number of messages (between 1 and 100), or all messages if "purge all" is specified
R.snipe - snipes a message from a channel or the current channel

R.2048 - play a 2048 game
R.hangman - play a Hangman game
R.matchpairs - play a Match Pairs game
R.minesweeper - play a Minesweeper game
R.rock-paper-scissors - play a Rock Paper Scissors game with your friend
R.snake - play a Snake game
R.tic-tac-toe - play a Tic Tac Toe game with your friend
R.trivia - play a Trivia game
R.worlde - play a Wordle game
